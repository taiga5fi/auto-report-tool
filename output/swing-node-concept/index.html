電力系統概念図解 自己学習用

スイングノードとは

潮流計算における「基準点」であり、誤差を吸収する場所ではない

用語の確認 → 直感 → 仕組み → よくある誤解 の順で積み上げる

先に用語をおさえる

この4語がわかれば後の説明が全部読める

母線（バス）とは？

変電所や発電所の「電気の接続ポイント」。ネットワークで言うノード（節点）にあたる。 複数の送電線・発電機・負荷がここでつながる。
= 電力系統のネットワーク上の「ノード」

位相角（θ）とは？

交流電気は電圧が波のように変化する（サイン波）。どの母線も同じ周波数で変化しているが、 波のピークのタイミングが微妙にズレている。この「ズレ具合」を角度（度またはラジアン）で表したのが位相角。
例：θ = 0°（基準）、θ = −15°（基準より少し遅れている）
重要： 位相角は「絶対値」ではなく「基準点からの相対値」なので、どこかに「θ = 0」の基準が必要。 ← これがスイングノードの役割

有効電力（P）と無効電力（Q）とは？

交流電気には2種類の電力がある。
有効電力 P：実際に「仕事」をする電力。電気機器を動かす。単位はW（ワット）。
無効電力 Q：コイルやコンデンサが蓄えて吐き出すだけで仕事をしない電力。電圧を安定させる役割。単位はvar（バール）。
= P が「仕事する電力」、Q が「電圧を保つための電力」

潮流とは？

系統内の各送電線に流れる電力（電気の流れ方）のこと。 「潮流計算」は、各母線の電圧・位相角を求めることで「どこにどれだけ電気が流れているか」を計算すること。
= 電力ネットワーク上の「トラフィック解析」に相当

まず直感でつかむ

2つの側面を別々のアナロジーで理解する

側面① 位相角の「基準点」としての役割
電子回路の「グラウンド（GND）」

回路の各ノードの電圧を計算するとき、どこか1点を「0V（グラウンド）」に固定します。 他のノードの電圧はこのゼロを基準にした相対値です。
グラウンドは「ノイズを吸収する場所」ではなく、「計算の出発点（ゼロ基準）」です。

回路解析： GND = 電圧 0V に固定した基準ノード
潮流計算： スイングノード = 位相角 θ = 0 に固定した基準母線
側面② 有効電力 P が「最後に決まる調整項」としての役割
会計の「差額を埋める調整項」

10人の食事代を計算するとき、9人分を先に決めて「残りは幹事が払う」とします。 幹事の金額は「全体の帳簿を閉じるために最後に決まる値」であり、「誤差や余りもの」ではありません。

会計： 幹事の負担額 = 9人が払った後の「残り」として決まる
潮流計算： スイングノードの P = 他の母線の P を全部決めた後の「残り」として決まる

→ これは誤差ではなく、系統全体の電力収支（発電 = 消費 + 損失）を成立させるための正当な解。

仕組みを理解する

潮流計算でスイングノードは何をしているか

潮流計算の目的

電力系統の各母線（接続ポイント）の電圧の大きさと位相角を求める計算。 これが分かると「どの送電線にどれだけ電力が流れているか」がわかる。

母線（ノード）の種類と「固定する値」「計算で求める値」の整理

※ ノード名は「固定する量の頭文字」から来ている（PQノード = P と Q を固定するからPQノード）

ノードの種類 固定する値（入力） 計算で求める値（出力）
PQノード
一般の負荷母線など		 有効電力 P（仕事する電力）
無効電力 Q（電圧を保つ電力）		 電圧 |V|（大きさ）
位相角 θ（ズレ角）
PVノード
発電機母線など		 有効電力 P
電圧の大きさ |V|		 無効電力 Q
位相角 θ
スイングノード
基準母線（1つだけ）		 電圧の大きさ |V|
位相角 θ = 0（これが「基準」)		 有効電力 P ← 調整項
無効電力 Q
ここが核心

スイングノードだけが θ を固定（= 0） することで「位相の基準点」を提供する。
その代わり、スイングノードの P・Q は固定しない。 全ノードの解が求まった後、系統全体の「発電 = 消費 + 損失」が成立するように P・Q が決まる。

計算の流れ（4ステップ）

1 初期値を仮に設定する

全母線の電圧・位相角に初期値を与える（スイングノードの θ = 0 だけは絶対に動かさない）

2 「今の解」でのズレを計算する

「今の電圧・位相角から計算した電力」と「指定した電力」のズレ（ΔP, ΔQ）を各母線で計算する。
→ これが「反復残差（潮流誤差）」。スイングノードに溜まるのではなく、全母線のズレが対象。

3 修正して繰り返す（収束するまで）

ズレが十分小さくなるまで繰り返す（「収束」）。収束しなければ解が得られない。スイングの θ は動かさない。

4 収束後：スイングノードの P・Q を求める

全母線の解（V, θ）が決まった後、スイングノードの P・Q を計算で求める。
→ このP は 「他の発電機・負荷の全合計との差（損失も含む）を埋める発電量」。誤差ではなく正当な解の一部。

よくある誤解

なぜその誤解が生まれるか、正しくはどう理解するか

誤解が生まれる理由： 「スイング（swing）」「スラック（slack = ゆるみ・余裕）」という名前から 「余分なものを受け取って消す場所」というイメージが生まれやすい。 また、スイングノードの P が「後から決まる」ことを「誤差が流れ込む」と読み違えやすい。
よくある誤解

「潮流計算で発生した誤差（潮流誤差）は、スイングノードで吸収・相殺されるから需給バランスは常に一致する」

正しくはこう理解する
正しい理解（2つに分けて考える）
①「反復残差（潮流誤差）」について

計算途中のズレ（ΔP, ΔQ）はスイングノードに「貼り付けられる」のではなく、 全母線のズレが繰り返し修正されてゼロに近づく（収束）のが正しい仕組み。 収束しなければ解全体が得られない。スイングノードが代わりに処理するわけではない。

②スイングノードのP（発電量）について

収束後に決まるスイングノードのPは「誤差」ではなく 「他の全発電機・全負荷の指定値と損失を合わせて、電力収支を成立させるために必要な発電量」。 会計で「9人の費用が決まった後に幹事が払う残り分」が「誤差」でないのと同じ。

補足：「スラック変数」との混同について（OPFを知っている方向け）

最適潮流計算（OPF）で使われる「スラック変数」（制約違反量を数値で表す変数）と、 潮流計算の「スラックノード（スイングノード）」は全く別の概念です。 どちらも「スラック」という言葉を使うため混同しやすいです。

潮流計算の「スラックノード」
θ・|V| を固定した基準母線。P・Q が後から決まる。母線の一種。
OPFの「スラック変数」
制約条件の違反量を表す数値変数（数理最適化の用語）。母線とは無関係。

人への説明まとめ

「スイングノードで誤差が吸収されるのでは？」への回答

30秒バージョン（そのまま使える）

「スイングノードは位相角 θ を 0 に固定した計算の基準点です。電子回路のグラウンドと同じです。
潮流計算では、スイングノードの θ・電圧は固定して他の値を求め、 最後に電力収支を閉じるためのスイングノードの発電量（P）が決まります。
これは誤差ではなく、系統全体の『発電 = 消費 + 損失』を成立させる正当な値です。
計算の途中の誤差（残差）はスイングノードに集まるのではなく、全母線で同時にゼロに収束させます。」

「潮流誤差はスイングノードで吸収されるから需給バランスは一致するのでは？」への回答
  • 「需給バランスが収束解で一致する」こと自体は正しい
  • 「スイングノードが誤差を吸収する」という理由づけは不正確
  • 正確な理由： スイングノードの発電量 P を「未指定のまま計算で決める」という定式化により、収束すれば系統の電力収支が自動的に成立するから
